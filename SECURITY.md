# Bezpieczeństwo

Jeśli znajdziesz podatność, **nie** publikuj szczegółów exploita w publicznym issue.
Skontaktuj się z maintainerami repozytorium **prywatnie** (np. [GitHub Security Advisories](https://docs.github.com/en/code-security/security-advisories), jeśli funkcja jest włączona dla tego repozytorium).
